About Me

I'm a data science and machine learning enthusiast currently juggling a B.Tech in Computer Science at VIT Vellore and a BS in Data Science from IIT Madras. I love turning raw data into smart solutions—be it classification models, full-stack dashboards or automation agents that speak plain English. My projects blend applied ML with a dash of creative chaos, from GenAI content generation with RLHF to interactive web apps that make ML more approachable. I code mostly in Python and Java (with coffee on standby) and have led object detection teams using YOLO and PyTorch. When I'm not building models, I help run events with the English Literary Association—because why not mix syntax with storytelling? I believe in clean code, explainable models, reproducible results, and yes, the occasional matplotlib overload
